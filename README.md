# rapidmotion

Rubymotion add ons

## Installation

Add this line to your application's Gemfile:

    gem 'rapidmotion'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rapidmotion

## Usage

# `UIViewController`
1. `tack`
2. `paste`
3. `shake`
4. `lift`



# `UIView`
1. `tack`
2. `paste`
3. `shake`
4. `cut`
5. `zoom_out_to`
5. `appear`
5. `vanish`
